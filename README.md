## Webpack 4 (4.10.2v) Vanilla Javascript Starter

- `npm install`
- `npm start` - developing
- `npm run build` - bundling to prod

### Includes:
- Hot reloading
- Dev / Prod separation builds
- Some build optimalizations (work still in progress)
- Sass
- Source maps on
- turned off side effects,
- babel-loader
- jpg / fonts bundling
